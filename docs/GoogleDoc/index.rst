事前準備
--------

共筆前，共筆成員需要準備以下幾項內容：

#. 申請一組Google帳號，以使用Google文件

#. 在Google文件中安裝GGEditor外掛程式，方便轉檔與推送

#. 申請一組Github帳號

#. 加入Github 專案共筆行列

Google帳號只需要到Google網站申請，因此不再贅述。剩下的2、3、4步驟流程，可參考以下文件：
